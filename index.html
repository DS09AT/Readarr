Download Shelvance

Windows

  1. Download the Windows package
  2. Extract the archive to your preferred location
  3. Run Shelvance.exe from the extracted folder
  4. Access the web UI at http://localhost:8787
Download for Windows

Linux

  1. Download the package for your architecture (x64, arm, arm64)
  2. Extract the archive: tar -xzf Shelvance*.tar.gz
  3. Run ./Shelvance from the extracted folder
  4. Access the web UI at http://localhost:8787
Download for Linux

macOS

  1. Download the .app.zip for your Mac (x64 or arm64)
  2. Extract and move Shelvance.app to Applications
  3. Right-click the app and select "Open" on first launch
  4. Access the web UI at http://localhost:8787
Download for macOS

Docker

  1. Pull the image: docker pull #COMING SOON#
  2. Create volumes for config and media paths
